# AI-Powered Tools

ShellGPT		- AI-powered tool that enhances accuracy in managing complex systems, writing code, and automating shell/command-line tasks.
AutoGPT         	- Automates task execution using autonomous AI agents.
WormGPT         	- Automates the generation of worm-like scripts and payloads.
ChatGPT         	- General-purpose conversational AI; also used with DAN prompt (Do Anything Now).
FreedomGPT      	- Privacy-focused AI chatbot alternative to ChatGPT.
FraudGPT        	- Designed to detect and prevent fraudulent activities.
ChaosGPT        	- Experimental AI agent with autonomous decision-making capabilities.
PoisonGPT       	- AI tool used to demonstrate data poisoning risks in AI models.
HackerGPT       	- AI assistant focused on offensive and defensive cybersecurity use cases.
BurpGPT         	- (https://www.burpgpt.app) AI integration with Burp Suite for pentesting assistance.
BugBountyGPT    	- AI tool designed for bug bounty hunters.
PentestGPT      	- AI assistant for penetration testing tasks.
GPTWhitehat     	- AI focused on ethical hacking and white-hat security practices.
CybGPT          	- AI assistant for general cybersecurity tasks.
BugHunter API  		- AI + API for bug hunting and vulnerability analysis.
h4ckGPT         	- Hacker-focused AI tool for automating red/blue team workflows.
HackerNewsGPT		- AI-powered summarizer and analysis of Hacker News content.
EthicalHackingGPT 	- AI assistant tailored for ethical hacking learning and practice.
GP(en)T(ester)  	- GPT-based penetration testing helper.
