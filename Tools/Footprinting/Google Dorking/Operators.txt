===========================
   GOOGLE SEARCH OPERATORS
===========================

üîπ Common Operators
----------------------
site:        - Search within a specific website or domain
intext:      - Find pages containing a specific word/phrase in the text
index:       - Look for directory-style listings
cache:       - View the cached version of a page
link:        - Find pages linking to a specific URL
related:     - Find sites related to a given URL
info:        - Show information about a specific website
allintitle:  - Search for pages with all given words in the title
intitle:     - Search for pages with specific word in the title
allinurl:    - Search for pages with all words in the URL
inurl:       - Search for pages with a specific word in the URL
location:    - Search for results from a specific location
filetype:    - Search for files of a specific type (e.g., PDF, DOCX, XLS)
source:      - Filter news results from a specific source
phonebook:   - Find phone listings (deprecated but may still work)
before:      - Results before a certain date (YYYY-MM-DD)
after:       - Results after a certain date (YYYY-MM-DD)

üîπ Image File Types
----------------------
jpg / jpeg   - Standard image format (most common photos)
png          - Lossless images with transparency
webp         - Modern compressed image format by Google
gif          - Animated or static images
bmp          - Bitmap format (uncompressed, large size)
tiff         - High-quality image format (used in design/printing)
svg          - Scalable Vector Graphics (XML-based, logos/icons)
ico          - Icon files (used in websites and software)
heic         - High Efficiency Image Format (iPhone photos)
raw          - Camera raw image data


üîπ Audio File Types
----------------------
mp3          - Most common compressed audio format
flac         - Lossless high-quality audio
wav          - Uncompressed audio format
wma          - Windows Media Audio
aac          - Advanced Audio Coding (used in iTunes/Apple)
m4a          - MPEG-4 audio (often Apple devices)
ogg          - Open-source compressed audio format
opus         - Audio codec optimized for speech/VoIP
aiff         - Audio Interchange File Format (Apple)
wax          - Windows Media redirect audio file


üîπ Subtitle / Script File Types
----------------------
srt          - SubRip subtitle file (timed captions)
ass          - Advanced SubStation Alpha subtitles (styled)
vtt          - Web Video Text Tracks (used in HTML5)
sub          - Subtitle format often paired with .idx


üîπ Video File Types 
----------------------
mp4          - Most common video format (web + devices)
mkv          - Matroska container (supports subs + multiple audio)
avi          - Older Microsoft video format
mov          - Apple QuickTime format
webm         - Open-source video format for the web
flv          - Flash Video (old streaming format)
wmv          - Windows Media Video
mpeg / mpg   - Standard video compression format
3gp          - Mobile video format
ogv          - Ogg Video container
m4v          - MPEG-4 video (used by iTunes/Apple)
vob          - DVD video object format
