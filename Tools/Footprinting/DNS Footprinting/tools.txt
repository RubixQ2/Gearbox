===========================
   DNS TOOLS & LOOKUP
===========================

üîπ Online DNS Lookup Tools
----------------------
MXToolbox           - https://mxtoolbox.com/                  - Lookup DNS records, MX, SPF, DKIM, and blacklist status.  
DNSstuff             - https://www.dnsstuff.com/              - Comprehensive DNS diagnostics and domain testing.  
ViewDNS.info         - https://viewdns.info/                  - Multi-tool suite including DNS, reverse DNS, and trace routes.  
WhatsMyDNS.net       - https://www.whatsmydns.net/           - Global DNS propagation checker for A, MX, CNAME, and TXT records.  
DNS Checker          - https://dnschecker.org/               - Check DNS records across multiple locations worldwide.  
CentralOps           - https://centralops.net/co/             - Domain/DNS research with host, traceroute, and WHOIS info.  

üîπ Command-Line / Recon DNS Tools
----------------------
DNSRecon             - https://github.com/darkoperator/dnsrecon      - Python tool for DNS enumeration and record discovery.  
Fierce               - https://github.com/mschwager/fierce           - DNS reconnaissance scanner for domain and subdomain discovery.  
Reverse IP Lookup    - https://www.yougetsignal.com/tools/web-sites-on-web-server/ - Identify all domains hosted on a given IP.  
PureDNS              - https://github.com/d3mondev/puredns           - Fast passive DNS resolver for subdomain enumeration.  
Bluto              - It retrieves DNS records, conducts zone transfers, and performs reconnaissance on target DNS servers.

üîπ DNS Security & Threat Intelligence
----------------------
SecurityTrails       - https://securitytrails.com/             - DNS history, subdomains, and threat intelligence.  
PassiveTotal         - https://www.riskiq.com/                 - Passive DNS collection and historical domain intelligence.  
Shodan               - https://www.shodan.io/                  - Search internet-connected devices and associated DNS/IP info.  
VirusTotal           - https://www.virustotal.com/gui/home/search - Check DNS records for malicious domains or IPs.  

üîπ DNS Automation & APIs
----------------------

DNSDB API            - https://www.farsightsecurity.com/solutions/dnsdb/ - Passive DNS database API for historical DNS research.
