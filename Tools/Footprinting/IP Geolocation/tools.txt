----------------------
IPinfo           - https://ipinfo.io/                     - Provides detailed IP data such as geolocation, ASN, hostname, and ISP info.  
MaxMind GeoIP    - https://www.maxmind.com/en/geoip2-services - Industry-standard IP geolocation database and API service.  
IP2Location      - https://www.ip2location.com/            - Offers IP geolocation, proxy detection, and time zone lookup.

DB-IP            - https://db-ip.com/                      - Provides IP geolocation API and database services.  
ipapi            - https://ipapi.co/                       - Real-time IP address lookup with location, ASN, and threat data.  
GeoIPTool        - https://geoiptool.com/                  - Simple tool to check geolocation, ISP, and IP information.  
WhatIsMyIP       - https://whatismyipaddress.com/ip-lookup - Shows IP, ISP, proxy detection, and geolocation details.

geoiplookup      - Linux CLI tool to query IP geolocation from databases like MaxMind.  
curl ipinfo.io   - Simple cURL query to fetch IP details from IPinfo API.  
ip2location-cli  - CLI client for querying IP2Location databases.
