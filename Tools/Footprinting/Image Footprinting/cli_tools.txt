# CLI Tools

exiftool 		- Command-line tool to extract metadata from images
mat2 			- CLI tool to remove metadata from images and documents for privacy
strings 		- Extract readable text from binaries and images
binwalk 		- Analyze images for embedded files or hidden data
steghide 		- Hide and extract data inside images or audio files
zsteg 			- Detect hidden data in PNG/BMP files for steganography
identify (ImageMagick) 	- Get image format, metadata, and properties
ffmpeg 			- Extract frames and metadata from videos
